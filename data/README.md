# data 폴더 구조 안내

이 폴더는 Streamlit 대시보드에서 사용하는 **최종 데이터 산출물**을 저장합니다.  
중간 결과물이나 실험용 데이터는 포함하지 않습니다.

---

## 폴더 규칙

- 폴더명 앞의 숫자는 **대시보드 흐름/페이지 순서**를 의미합니다.
- `00_common`은 특정 페이지에 속하지 않는 **공통 기준 데이터**입니다.
- 각 폴더에는 해당 페이지에서 직접 사용하는 데이터만 둡니다.

---

## 폴더 설명

### 00_common
- 모든 페이지에서 공통으로 참조하는 기준 데이터
- 예:
  - hit_score
  - master_meta (작품 단위 메타 정보)
  - lookup 테이블 (장르, 국가 등)

---

### 01_overview
- Overview 페이지 전용 집계 결과
- 예:
  - 연도별 트렌드
  - Top N 작품 요약

---

### 02_synopsis_tfidf
- 줄거리(Synopsis) 기반 TF-IDF 분석 결과
- 예:
  - 흥행/비흥행 키워드
  - 타입별 키워드 비교

---

### 03_bertopic
- BERTopic 분석 결과
- 예:
  - 토픽 요약
  - 토픽별 흥행 지표

---

### 04_review
- 리뷰 데이터 분석 결과
- 예:
  - 리뷰 TF-IDF 키워드
  - 리뷰 토픽 요약 (전체 / 타입별)

---

### 05_prediction
- 예측 및 추천 모델 결과
- 예:
  - 흥행 예측 결과
  - 유사 작품 추천 리스트

---

## 주의사항

- 이 폴더에는 **대시보드에서 직접 읽는 파일만** 둡니다.
- 중간 산출물, 실험 파일은 `outputs/`, `tmp/` 등 별도 폴더를 사용합니다.
- 파일명은 `[도메인]_[내용]_[스코프].parquet` 규칙을 따릅니다.
